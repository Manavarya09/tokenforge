# Contributing

## Quick start
```bash
git clone https://github.com/Manavarya09/tokenforge
cd tokenforge
cargo build
```

## PR checklist
- [ ] Branch from `main` (e.g. `feat/...`, `fix/...`, `chore/...`)
- [ ] `cargo test` passes
- [ ] No leftover `dbg!()` or `println!()` debugging
- [ ] Commit messages explain *why*

## Reporting bugs
Open an issue with steps + expected vs actual. Security: see SECURITY.md.
